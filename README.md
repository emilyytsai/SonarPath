# SonarPath
SonarPath was developed during the hackathon, **FullyHacks 2026**, at CSUF and won **Best Social Impact & Sustainability**!

SonarPath is a real-time sea routing tool that helps ships avoid whale habitats by visualizing their underwater noise impact. It optimizes routes to protect marine life, while also reducing fuel costs and providing eco-friendly incentives.
